# Nutrifit - Nutrition and Fitness Tracking App

## Description
### Python Jackfruit Problem:

A simple nutrition and fitness tracking app made in python using PyQt6.

## Installation
Clone this repo using: 
```
git clone https://github.com/jackfroot30/nutrifit.git
cd nutrifit
```
Create a virtual environment and activate it:
```
python -m venv virtualenv
source virtualenv/bin/activate # for Linux & MacOS
.\virtualenv\Scripts\activate # for Windows
```
Install Dependencies:
```
pip install -r requirements.txt
cd src_1
```
Run:
```
python main.py  
```
## Usage
An API key from [https://aistudio.google.com/app/api-keys](https://aistudio.google.com/app/apikey) is required for the working of the app. Store the API key in the format `API_KEY=<your api key>` in the same folder as the app.

First, enter the src_1 directory using:
```
cd src_1
```
Run the app using:
```
python main.py
```
